Organisation Identifiers

The IATI Technical Advisory Group is involved in ongoing work to establish a set of principles to uniquely identify any organisation in the world, be it government or NGO, public or private. Identifiers should generally be based upon existing nationally issued identifiers and should use an agreed namespace code prefix. Please consult the Organisational Identifiers Guidance to identify the appropriate codes to use in your data. The code list below is provided for use when existing organisational identifiers are not available.
