Schema Changes

A full diff of all schema changes in 2.02 can be found in the IATI GitHub

Activities schema

Addition of @humanitarian attribute within the iati-activity element. (Discussion)

Addition of @activity-id attribute within the participating-org element. (Discussion)

Addition of @vocabulary-uri attribute within the recipient-region element. (Discussion)

Addition of @vocabulary-uri attribute within the sector element. (Discussion)

The element humanitarian-scope was added (Discussion)

Addition of @vocabulary-uri attribute within the policy-marker element. (Discussion)

The occurrence rule for the @significance attribute within the policy-marker element was amended. (Discussion)

Addition of @status attribute within the budget element. (Discussion)

The element provider-org (a sub-element of planned-disbursement) was added (Discussion)

The element receiver-org (a sub-element of planned-disbursement) was added (Discussion)

The definition for the element period-start (a sub-element of planned-disbursement) was amended (Discussion)

The definition for the element period-end (a sub-element of planned-disbursement) was amended (Discussion)

Addition of @humanitarian attribute within the transaction element. (Discussion)

Addition of @type attribute within the provider-org element (itself a sub-element of transaction). (Discussion)

Addition of @type attribute within the receiver-org element (itself a sub-element of transaction). (Discussion)

Addition of @vocabulary-uri attribute within the transaction/sector element. (Discussion)

Addition of @vocabulary-uri attribute within the transaction/region element. (Discussion)

The element document-date (a sub-element of document-link) was added (Discussion)

The element reference (a sub-element of result/indicator) was added (Discussion)

The element location (a sub-element of result/indicator/period/target/) was added (Discussion)

The element dimension (a sub-element of result/indicator/period/target/) was added (Discussion)

The element location (a sub-element of result/indicator/period/actual/) was added (Discussion)

The element dimension (a sub-element of result/indicator/period/actual/) was added (Discussion)

The element channel-code a sub-element of crs-add/) was added (Discussion)

Organisation schema

Addition of @status attribute within the total-budget element. (Discussion)

Addition of @status attribute within the recipient-org-budget element. (Discussion)

The element recipient-region-budget was added (Discussion)

The element total-expenditure was added (Discussion)

The element document-date (a sub-element of document-link) was added (Discussion)

Codelist schema

Within the raw codelist XML definitions, each codelist-item now supports the addition of status, activation-date and withdrawal-date attributes. (Discussion)
