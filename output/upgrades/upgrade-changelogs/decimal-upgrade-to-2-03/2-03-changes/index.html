Schema Changes

A full diff of all schema changes in 2.03 can be found in the IATI GitHub.

These changes are also shown in the Final Technical Summary (spreadsheet).

Activities Schema

  • Added new @crs-channel-code attribute for the participating-org element (#338) (discussion link)

  • Added new tag element (as a child of the iati-activity element) (#342) (discussion link)

  • Updated definition: [recipient-country, recipient-region, sector] @percentage attribute (#343) (discussion link)

  • Updated definition: sector @vocabulary attribute (#344). The % sign was removed.

  • Updated definition: [transaction/aid-type, default-aid-type] @code attribute (#345) (discussion link)

  • Added new @vocabulary attributes for elements relating to aid-type (#346) (discussion link)

  • Updated definition: capital-spend @percentage attribute (#348) (discussion link)

  • Updated definition: related-activity element (#349) (discussion link)

  • Updated occurrence: default-aid-type element (#350) (discussion link)

  • Updated occurrence: transaction/aid-type element (#351) (discussion link)

Changes and Additions to the Result element:

  • Updated definition: result/indicator/reference @indicator-uri attribute (#347) (discussion link)

  • Updated occurrence: result/indicator/baseline element (#352) (discussion link)

  • Updated occurrence: result/indicator @value attributes (#353) (discussion link)

  • Updated occurrence: [result/indicator/period/target, result/indicator/period/actual] elements (#354) (discussion link)

  • Updated occurrence: result/indicator/period/target @value attribute (#355) (discussion link)

  • Updated occurrence: result/indicator/period/actual element (#356) (discussion link)

  • Updated occurrence: result/indicator/period/actual @value attribute (#358) (discussion link)

  • Added new document-link element (as a child of various result elements)(#359) (discussion link)

  • Added new reference element (as a child of the result element) (#362) (discussion link)

  • Added new @aggregation-status attribute for the result/indicator element (#363) (discussion link)

  • Added new location element (as a child of the result/indicator/baseline element) (#365) (discussion link)

  • Added new dimension element (as a child of the result/indicator/baseline element) (#366) (discussion link)
