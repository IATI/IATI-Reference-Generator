Country Budget Items

<country-budget-items></country-budget-items>

From the schema

Recipient country budget items. This item encodes the alignment of activities with both the functional and administrative classifications used in the recipient country's Chart of Accounts. This applies to both on- and off-budget activities.

@vocabulary A code for the common functional classification or country system (This allows for common codes, country-specific, or any other classification agreed between countries and donors).

Standard compliance test(s) for /country-budget-items/@vocabulary

  • Must be on: Budget Identifier Vocabulary Codelist
  • Must be on the specified codelist given by country-budget-items/@vocabulary.
    If country-budget-items/@vocabulary = 1 (i.e. IATI) then must be on: Budget Identifier Codelist

Country Budget Items

Example Usage

<country-budget-items vocabulary="XXX">
    <budget-item code="XXX" percentage="100">
       <description xml:lang="en">XXX</description>
    </budget-item>
</country-budget-items>
