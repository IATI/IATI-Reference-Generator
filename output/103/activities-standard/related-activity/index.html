XX.XX Related Activity Another IATI activity related to this one. The 'type' attribute describes the type of relationship (e.g. parent, sibling). This does not need to be used to express funding relationships, since those are covered in individual transactions. For the value of the @type attribute, see http://iatistandard.org/codelists/related_activity_type

@ref Machine-readable identification string for the business object being described.

Related Activity

Example Usage

The reference and type of relation can be cited in this element: 

   <related-activity type="1" ref="GB-1-105838"/>

Additionally, a text title of the specific related activity can be provided: 

    <related-activity type="1" ref="GB-1-105838">Trade Sector Programme</related-activity>

Where this text is in a language that differs from the default set in <iati-activity>, then this should be declared accordingly: 

    <related-activity type="1" ref="GB-1-105838" xml:lang="fr">Programme du secteur commercial</related-activity>
