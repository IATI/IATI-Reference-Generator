Technical Issues

There are a number of technical issues that an organisation may need to consider when working towards IATI publication.

These include looking at how data is currently collected and stored and how easy it is to pull together (e.g. whether it can automatically feed into an IATI reporting system or if it needs to be pulled from different sources).

Another consideration is around in-house capacity and expertise (i.e. whether an organisation has the ability to convert their data into IATI format using internal systems or technical capacity), or whether third party tools need to be used.
