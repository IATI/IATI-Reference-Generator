Optional Data

There are currently three elements in the IATI Standard which are considered optional. These are:

  • Sub-national Geographic Location
  • Text of Conditions
  • Results data

These remain optional at present as, although some organisations are publishing information under these areas, it is recognised that not all organisations capture this data, or at least not in a standardised, easily accessible format. Organisations may need to do additional work or change their data capture approach in order to capture and/or report this type of data.

The aim however, is for organisations to publish information relevant to these areas at some point in their future publications if it is not already available.
