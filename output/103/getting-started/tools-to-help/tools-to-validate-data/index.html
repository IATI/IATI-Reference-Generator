Tools to Validate Data

IATI Public Validator

For those wishing to validate IATI XML files, a tool is available at: http://validator.iatistandard.org/

This tools enables people to add data for testing via upload, web address or simply pasting the code into the browser.

The resultant data is then tested for:

  • Well-formed XML – a simple document test
  • IATI Activity Schema – validation against the latest schema.

Results of these tests are provided onscreen, alongside further information and analysis of the supplied data.

The source code for this tool is also available at GitHib.

IATI Data Spotter

For those wishing to analyse and validate their datasets, the IATI Data Spotter is a project to assist.

This tool enables people to run various checks and queries over a dataset, in order to understand and identify any potential discrepancies.

The source code for this project is available to download and use. Contributions are also welcome.
