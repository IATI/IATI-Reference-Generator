Tools to Create Data

There are a two third-party tools available for use in converting / publishing IATI data.

Web-entry platforms

There are currently two web-entry platforms that can be used to publish data:

  • AidStream: http://aidstream.org/public/
  • Open Aid Register: http://www.openaidregister.org/

The web-entry platforms act as online project management systems or data bases, whereby information is entered manually in the form or text-based information or selecting responses from drop-down menus. This covers all of the activity information required by IATI and will automatically convert the data into XML and provide the appropriate links to register the files on the IATI registry.

Spreadsheet conversion tool

The spreadsheet conversion tool can be used to convert data stored in a CSV file by mapping the columns in the data to IATI fields. The tool will pull out the information and convert it into XML data. This can also be used for organisations reporting on CRS data, as long as it is captured in a spreadsheet.
There are certain preconditions for using the tool: the data must use IATI codelists, be formatted according to IATI Standard specifications (e.g., dates must be in the format YYYY-MM-DD) and be contained in a single worksheet.
The tool also has several limitations. It does not permit a hierarchy of activities, or multiplicity within more than one dimension (e.g. multiple sectors and multiple recipient countries).
Finally, the tool does not currently provide any verification of compliance with the Standard.
You can access the tool here: http://csv2iati.iatistandard.org/
