Organisation Identifier

Definitions

The organisation-identifier element uniquely identifies the organisation being reported.

iati-identifier-text A unique identifier for the organisation that is consistent with IATI’s rules for identifying organisations.

Example Usage

 

  ..
  US-EIN-12345
  ..

Structure

Item Format XML Occurs
iati-identifier iati-identifier 1..1
   iati-identifier-text see IATI Rules for Organisation Identifiers. iati-identifier/text() 1..1

Usage & Compliance

This element is mandatory.

  • The iati-identifier-text is mandatory.
  • The format of iati-identifier-text MUST comply with IATI Rules for Organisation Identifiers.
