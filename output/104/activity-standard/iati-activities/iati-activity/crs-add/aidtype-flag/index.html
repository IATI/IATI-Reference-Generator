aidtype-flag

iati-activities/iati-activity/crs-add/aidtype-flag

This is the reference page for the XML element aidtype-flag.

Definition

This covers the four CRS++ fields titled: “Free standing technical cooperation”; “Programme-based approach”; “Investment project”; “Associated financing”

Attributes

@code

Machine readable code for the entity being described. The value should be available on a related codelist.

This value must be of type xsd:string.

This value must be on the AidTypeFlag codelist.

@significance

Does this flag apply? If ‘false’ do not report the flag

This value must be of type xsd:boolean.

Example Usage

Example usage of aidtype-flag in context of crs-add element.

Example declares AidTypeFlag code 1 and significance through optional boolean.

  <crs-add>
   <aidtype-flag code="1" significance="1" />
   <loan-terms rate-1="4" rate-2="3">
    <repayment-type code="1" />
    <repayment-plan code="4" />
    <commitment-date iso-date="2013-09-01"/>
    <repayment-first-date iso-date="2014-01-01" />
    <repayment-final-date iso-date="2020-12-31" />
   </loan-terms>
   <loan-status year="2014" currency="GBP" value-date="2013-05-24">
     <interest-received>200000</interest-received>
     <principal-outstanding>1500000</principal-outstanding>
     <principal-arrears>0</principal-arrears>
     <interest-arrears>0</interest-arrears>
   </loan-status>
  </crs-add>
