transaction-date

iati-activities/iati-activity/transaction/transaction-date

This is the reference page for the XML element transaction-date.

Definition

A milestone date for this transaction (such as the decision date, disbursement date, etc.).

Attributes

@iso-date

The ISO 8601 version of the transaction date.

This value must be of type xsd:date.

transaction-date/@iso-date must not be in the future.

Example Usage

Example usage of transaction-date in context of transaction element.

A date in ISO 8601 format (YYYY-MM-DD) for the transaction-date is required:

  <transaction ref="1234">
   <value currency="EUR" value-date="2012-01-01">1000</value>  
   <description>Transaction description text</description>
   <transaction-type code="IF" />
   <provider-org provider-activity-id="BB-BBB-123456789-1234AA" ref="BB-BBB-123456789">Agency B</provider-org>
   <receiver-org receiver-activity-id="AA-AAA-123456789-1234" ref="AA-AAA-123456789">Agency A</receiver-org>
   <transaction-date iso-date="2012-01-01" />
   <disbursement-channel code="1" />
  </transaction>

Full example with additional elements which can override the default value set in iati-activity:

  <transaction ref="1234">
   <value currency="EUR" value-date="2012-01-01">1000</value>   
   <description>Transaction description text</description>
   <transaction-type code="IF" />
   <provider-org provider-activity-id="BB-BBB-123456789-1234AA" ref="BB-BBB-123456789">Agency B</provider-org>
   <receiver-org receiver-activity-id="AA-AAA-123456789-1234" ref="AA-AAA-123456789">Agency A</receiver-org>
   <transaction-date iso-date="2012-01-01" />
   <flow-type code="20" />
   <finance-type code="111" />
   <aid-type code="A02" />   
   <tied-status code="5" />
   <disbursement-channel code="1" />
  </transaction>
