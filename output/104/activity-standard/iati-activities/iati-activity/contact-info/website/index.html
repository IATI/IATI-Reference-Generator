website

iati-activities/iati-activity/contact-info/website

This is the reference page for the XML element website.

Definition

The contact web address, if available. May be repeated for multiple sites.

The text in this element must be of type xsd:anyURI.

Example Usage

Example website within contact-info containing all possible child elements:

  <contact-info type="1">
   <organisation>Agency A</organisation>
   <person-name>A. Example</person-name>
   <job-title>Transparency Lead</job-title>
   <telephone>0044111222333444</telephone>
   <email>transparency@example.org</email>
   <mailing-address>Transparency House, The Street, Town, City, Postcode</mailing-address>   
   <website>http://www.example.org</website>
  </contact-info>
