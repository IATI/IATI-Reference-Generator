default-aid-type

iati-activities/iati-activity/default-aid-type

This is the reference page for the XML element default-aid-type.

Definition

The type of aid being supplied (budget support, debt relief, etc.). This element specifies a default for all the activityâ€™s financial transactions; it can be overridden at the individual transaction level.

For the value of the @code attribute, see http://iatistandard.org/codelists/aid_type

Attributes

@code

Machine readable code for the entity being described. The value should be available on a related codelist.

This value must be of type xsd:string.

This value must be on the AidType codelist.

@xml:lang

ISO 2 letter code specifying the language of text in this element.

This value should be on the Language codelist.

Example Usage

Example default-aid-type declaring the AidType code A01 (General budget support):

  <default-aid-type code="A01" />
