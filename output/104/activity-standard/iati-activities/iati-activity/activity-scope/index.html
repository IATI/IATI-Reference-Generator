activity-scope

iati-activities/iati-activity/activity-scope

This is the reference page for the XML element activity-scope.

Definition

What geographical area does the activity encompass? eg. Global, Regional, Multi-National, National, Multiple (sub-national) administrative areas, etc

Attributes

@code

Machine readable code for the entity being described. The value should be available on a related codelist.

This value must be of type xsd:string.

This value must be on the ActivityScope codelist.

@xml:lang

ISO 2 letter code specifying the language of text in this element.

This value should be on the Language codelist.

Example Usage

Example of an activity-scope with ActivityScope code 3 (Multi-national):

  <activity-scope code="3" />
