Tools to View Data

Viewing and accessing IATI data can be undertaken through the following sources:

IATI Registry

The IATI Registry is the central resource of links to all data published to the IATI standard. Via the Registry it is possible to search datasets by publisher, type and country. When using the Registry, it is also possible to “preview” individual data sets. On accessing this, a version of the data is made available, that is accessible within a web browser.

IATI Explorer

The IATI Explorer is a tool provided by Development Gateway. This tool enables people to search, query, access and download data across IATI.

AidView Beta

AidView visualises IATI datasets via an interactive “app”. This enables people to search and query IATI data from multiple viewpoints, and access the data as a graph, list or map.
