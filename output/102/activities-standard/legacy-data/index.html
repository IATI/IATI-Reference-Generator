Legacy Data

<legacy-data></legacy-data>

From the schema

Can occur: Min = 0, Max = Unlimited

Hold a single name=value pair of legacy data. This element is *not* for adding new data types; instead, it holds the original (non-IATI) value or code for an existing data type.

@name The original field name.
@value The original field value.
@iati-equivalent The name of the equivalent IATI element (if available).
