Classifications

Although a sector(s) should be provided for each activity, it is recognised that not all organisations use the same sector definitions. As such, this is one element within IATI whereby an organisation can choose to report using an existing set of sectors (for which code lists are provided) or can define their own sectors if these do not map easily across.

This section provides guidance on how to report sectors and also types of information which may not fit within default IATI elements.
