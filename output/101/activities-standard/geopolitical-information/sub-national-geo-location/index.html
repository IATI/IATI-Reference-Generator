Sub-national Geographic Location

<location></location>

From the schema

A geographical location.

@percentage The percentage of the activity budget allocated to this location.

Standard compliance test(s) for /location

  • location has a name
  • percentages of locations sum to 100
  • Must be on: Country Codelist
  • Must be on: Geographical Precision Codelist
  • Must be on: Gazetteer Agency Codelist
  • Must be on: Location Type Codelist

Location

Example Usage

The location element is used a contain various other elements 

    <location>
       ....
    </location>

If more than one location is reported, percentage of activity commitment allocated to this location (if available). 

    <location percentage="85>
       ....
    </location>
    <location percentage="15>
       ....
    </location>

Example from UNOPS: 

<location>
      <name>Herat</name>
      <coordinates latitude="34.341944400000003000" longitude="62.203055599999971000" precision="2" />
      <location-type code="PPL" />
      <administrative country="AF">Afghanistan, Herat, Injil</administrative>
    </location>
