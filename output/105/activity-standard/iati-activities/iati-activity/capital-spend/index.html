capital-spend

iati-activities/iati-activity/capital-spend

This is the reference page for the XML element capital-spend.

Definition

The percentage of the total commitment that is for capital spending

Attributes

@percentage

The percentage of the budget allocated to this item. Content must be a positive decimal number between 0 and 100, with no percentage sign. In version 1.03 of the IATI Standard this value became designated as a decimal value and no longer as a positive integer.

This value must be of type xsd:decimal.

Example Usage

Example capital-spend that uses percentage to state 88.8% of the iati-activity budget is for capital expenditure:

  <capital-spend percentage="88.8" />
