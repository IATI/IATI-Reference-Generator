activity-status

iati-activities/iati-activity/activity-status

This is the reference page for the XML element activity-status.

Definition

The current status of the project (e.g. “planned”), using a list defined by IATI. For the value of the @code attribute, see http://iatistandard.org/codelists/activity_status

Attributes

@code

Machine readable code for the entity being described. The value should be available on a related codelist.

This value must be of type xsd:string.

This value must be on the ActivityStatus codelist.

@xml:lang

ISO 2 letter code specifying the language of text in this element.

This value should be on the Language codelist.

Example Usage

Example of an activity-status with ActivityStatus code 2 (Implementation):

  <activity-status code="2" />
