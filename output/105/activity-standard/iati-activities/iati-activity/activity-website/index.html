activity-website

iati-activities/iati-activity/activity-website

This is the reference page for the XML element activity-website.

Definition

A direct link to a web site or web page providing more information about the specific aid activity. Multiple links can be included. Multiple versions of the link may appear for different languages. Should not be general websites or addresses, and should include the http or https prefix.

The text in this element must be of type xsd:anyURI.

Example Usage

Example of an activity-website which provides a URL to the webpage for a specific iati-activity:

  <activity-website>http://www.example.com/en/activity/ABC123</activity-website>
