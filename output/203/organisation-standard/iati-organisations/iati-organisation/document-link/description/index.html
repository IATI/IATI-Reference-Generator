description

iati-organisations/iati-organisation/document-link/description

This is the reference page for the XML element description.

Definition

A description of the document contents, or guidance on where to access the relevant information in the document.

Rules

This element must occur no more than once (within each parent element).

Example Usage

Example usage of description of a document-link in an iati-organisation.

 <document-link format="application/vnd.oasis.opendocument.text" url="http://www.example.org/docs/report_en.odt">
  <title>
    <narrative>Annual Report 2013</narrative>
   </title>
   <description>
    <narrative>Description of the annual report.</narrative>
    <narrative xml:lang="fr">Description du rapport annuel.</narrative>
  </description>
   <category code="B01" />
   <language code="en" />
   <document-date iso-date="2014-02-05" />
   <recipient-country code="AF">
    <narrative>Afghanistan</narrative>
   </recipient-country>
  </document-link>
